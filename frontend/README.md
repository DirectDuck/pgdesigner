# PgDesigner Frontend

Vue 3.5 + TypeScript + Vue Flow ERD canvas with Go backend (zenrpc JSON-RPC).

## Quick Start

```bash
# Full build (frontend + Go binary)
make build-full
make run-pagila             # 15 tables (partitions, triggers, views)
make run-adventureworks     # 68 tables (5 schemas)

# Development (two terminals)
make dev-backend        # Go backend on :9990
make dev-frontend       # Vite dev server on :5173 (proxy /rpc → :9990)
```

## ERD Canvas Interaction

### Mouse

| Action | Effect |
|--------|--------|
| **Drag on empty area** | Pan (move canvas) |
| **Shift + drag on empty** | Rubber-band selection (select multiple tables) |
| **Click table** | Select table |
| **Shift + click table** | Add/remove from selection |
| **Drag table** | Move table (all selected move together) |
| **Scroll wheel** | Pan vertically |
| **Shift + scroll** | Pan horizontally |
| **Pinch (trackpad)** | Zoom in/out |

### Keyboard

| Shortcut | Action |
|----------|--------|
| **Ctrl+F** | Go To (spotlight search) |
| **?** | Keyboard reference |
| **Enter / F2** | Edit selected table |
| **↑↓←→** | Navigate columns/items in editor |
| **Space** | Toggle boolean fields |
| **+/−** | Add/delete items in editor |
| **Ctrl+D** | Diff unsaved changes |
| **Ctrl+S** | Save to disk |

### Toolbar

| Button | Action |
|--------|--------|
| **Zoom +** | Zoom in |
| **Zoom -** | Zoom out |
| **%** (click) | Reset zoom to 100% |
| **Fit** | Fit all tables to viewport |
| **Fix Overlaps** | Push overlapping tables apart, preserve layout |
| **Auto Layout** | Sugiyama hierarchical layout + fix overlaps + fit |

### Minimap

Located in bottom-right corner. Pannable and zoomable — drag to navigate.

## Tech Stack

- Vue 3.5 + TypeScript + Composition API + Pinia 3
- Vue Flow (@vue-flow/core) — ERD canvas (DOM-based, 120+ tables)
- Reka UI — splitter, menubar, dialog
- Tailwind CSS v4
- Iconify — icons
- html-to-image — PNG/SVG export
- CodeMirror 6 — SQL viewer
- Vite 7 + pnpm

## Theming

Light and dark themes via CSS custom properties.

1. All colors are CSS custom properties in `src/assets/main.css` (`:root` light, `.dark` dark)
2. No hardcoded hex colors in components — use `var(--color-*)`
3. Semantic token names — `--color-bg-app`, `--color-text-primary`, `--color-table-border`
4. ERD icons keep pgmdd colors in both themes (gold PK, gray FK, red NN, green index)
5. Toggle: View → Dark Theme. State in `localStorage('pgd-theme')`

| Prefix | Purpose |
|--------|---------|
| `--color-bg-*` | Background (app, surface, panel, hover, selected) |
| `--color-text-*` | Text (primary, secondary, muted, disabled) |
| `--color-border*` | Borders (default, subtle, strong) |
| `--color-table-*` | ERD table card (bg, border, header, row hover) |
| `--color-icon-*` | ERD icons (pk, fk, nn, null, index, table) |
| `--color-line-*` | FK relationship lines (default, gray, markers, labels) |
| `--color-minimap-*` | Minimap (bg, table fill/stroke, lines) |
| `--color-menu-*` | Menu/dropdown (bg, border) |

## RPC API

Backend serves JSON-RPC at `/rpc/`. SMDBox docs: `/rpc/doc/`.

Key namespaces: `project.*`, `app.*`, `collection.*`.

## Build

```bash
pnpm install            # install dependencies
pnpm dev                # dev server (needs Go backend on :9990)
pnpm build              # production build → dist/
pnpm lint               # oxlint + eslint
pnpm test:unit          # vitest
```

Frontend is embedded into Go binary via `go:embed` (see `embed.go`).
